# Despliegue Backend

Este es el backend del proyecto de Fútbol Colombiano. Está construido con **Node.js** y **Express**, y se conecta a una base de datos **MySQL**.

## Requisitos
- Node.js
- MySQL (Workbench o XAMPP)

## Instalación
1. Clona el repositorio.
2. Ejecuta `npm install` para instalar las dependencias.
3. Importa el archivo `database.sql` en tu base de datos MySQL.
4. Configura las credenciales en `server.js` (o variables de entorno si las usas).
5. Inicia el servidor con `npm start`.

El servidor correrá en `http://localhost:3000`.
